# Lektion 1: Einführung in Datenvisualisierung

## Überblick

**Format:** Hybrid (Präsenz + Online)
**Lernziele:**
1. Unterschied zwischen Matplotlib und Seaborn verstehen
2. Edward Tufte's Prinzipien anwenden können
3. Barrierefreie Visualisierungen erstellen
4. Erste eigene Plots programmieren

## Hausaufgabe

**Aufgabe:** Erstelle eine Visualisierung deiner Wahl, die:
1. Matplotlib ODER Seaborn verwendet
2. Tufte-Prinzipien befolgt (hohe Data-Ink Ratio)
3. Barrierefrei ist (WCAG AA)
4. Gut dokumentiert ist (Kommentare im Code)

**Abgabe:** Als Jupyter Notebook

**Bewertungskriterien:**
- Technische Umsetzung (40%)
- Tufte-Prinzipien (30%)
- Barrierefreiheit (20%)
- Dokumentation (10%)
